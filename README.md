# 📊 Online Course Sales Data Analysis Dashboard

An interactive **Power BI Desktop** dashboard analyzing **8,092 online courses** across multiple e-learning platforms, built to uncover trends in course distribution, pricing, ratings, and learner engagement.

---

## 🔍 Overview

This project explores a large-scale dataset of online courses (Coursera and other platforms) to help understand:

- How courses are distributed across categories, sub-categories, and languages
- Which course types (Course / Specialization / Project) attract the most viewers
- Pricing structures and availability trends
- Popular learning domains and educational opportunities
- Engagement metrics such as ratings, registrations, and viewership

---

## 🎯 Dashboard Purpose

The dashboard enables users to:

- ✅ Analyze course distribution across platforms
- ✅ Compare course categories and languages
- ✅ Evaluate pricing trends and course availability
- ✅ Identify popular learning domains and educational opportunities

---

## 🗂️ Dataset

| Detail | Description |
|---|---|
| **Source file** | `Online_Courses__1_.csv` |
| **Total records** | 8,092 online courses |
| **Platforms** | Coursera and other learning platforms |
| **Key fields** | Title, URL, Category, Sub-Category, Course Type, Language, Subtitle Languages, Skills, Instructors, Rating, Number of Viewers, Duration, Price, Level, Number of Reviews, and more |

---

## 📈 Key Insights & Visuals

### 1. Course Performance, Registrations & Learner Engagement
- **Total Courses:** ~3K
- **Average Rating:** 4.70 ⭐
- **Overall Registrations:** 8M
- **Average Registrations per Course:** 3.21K
- Breakdown of total and average viewers by **Course Type** (Course, Specialization, Project)

### 2. Average Viewers by Category
- **Data Science** leads with the highest average viewers (6.3K), followed by **Information Technology** (4.6K) and **Computer Science** (4.4K)

### 3. Language Distribution by Category
- Sub-category and course-count breakdown across categories
- Identifies dominant sub-categories such as Data Analysis, Software Development, and Business Essentials

### 4. Category Ranking by Number of Views
- Ranked table of categories by average viewers, drilled down by language (English, Spanish, French, Japanese, Portuguese, etc.)
- **Data Science** ranks #1 overall with 6,326.70 average viewers

### 5. Views by Number of Available Languages
- Explores how course reach (in available languages) correlates with average viewership
- Courses available in ~27 languages show a sharp spike in average viewers (86K)

### 6. Tutors & Duration
- Total tutor count (~4K) and category-wise tutor rankings
- Distribution of viewers by course duration (hours)
